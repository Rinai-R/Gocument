# Gocument

**Gocument** 是一个**在线文档协作平台**，目前正在完善中...

## 实现功能

- [x] 用户注册登陆
- [ ] 登陆状态创建/编辑文档
- [ ] 文档存储于数据库/存储优化
- [ ] 自动保存(高性能)
- [ ] 文档分享/文档的私有和公开
- [x] 个人主页(查看我的文档及其基本信息)
- [ ] 删改文档
